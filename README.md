Music Player
====

このパッケージはミュージックプレイヤーを提供するSwiftUI製のパッケージです。

## Description

iOSの端末のAppleMusicの楽曲を再生します。

機能一覧はこちら。

* 楽曲の再生、一時停止、次の曲の再生、前の曲の再生
* 曲の再生速度の変更
* 曲のキーの高さの変更
* 再生時間の変更(シークバーを想定)
* 曲のバックグラウンド再生
* 画面オフ状態での曲の操作(MPRemoteCommandCenterの使用)
* 電話やイヤホンなどの割り込みによる曲の再生、一時停止処理
